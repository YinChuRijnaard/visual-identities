# Style Guide

This document contains the general styling rules for the website of Visual Identities.

### Typography

| Element     | `sm`        | `lg` |
| ----------- | ----------- | ---- |
| Text small  | `text-sm`   |      |
| Text base   | `text-base` |      |
| Heading 1   | `text-2xl`  |      |
| Heading 2   | `text-xl`   |      |
| Heading 3   | `text-lg`   |      |
| Button (sm) | `text-base` |      |
| Button (lg) | `text-lg`   |      |
| Logo        | `text-xl`   |      |
| Hero        | `text-2xl`  |      |

### Colours

| Element             | Colour             |
| ------------------- | ------------------ |
| Background (base)   | `bg-neutral-100`   |
| Background (darker) | `bg-neutral-200`   |
| Text (base)         | `text-neutral-900` |
| Text (darker)       | `text-neutral-500` |

### Margin

|                        | `sm`   | `lg`    |
| ---------------------- | ------ | ------- |
| Horizontal margin      | `mx-4` | `mx-32` |
| Between components (y) | `my-8` |         |
| Between text (y)       | `my-2` |         |

### Padding

|         |       |
| ------- | ----- |
| Element | `p-4` |
